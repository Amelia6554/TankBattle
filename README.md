# TankBattle

Defeat all enemies to win. You can also destroy obstacles on your way.  
1 project on 3 platforms.

---

## How to play?

### Desktop
- Download `.exe` file  
- Controls: `W`, `A`, `S`, `D` to move, LMB to shoot, `ESC` to pause  
- Features: save system, sounds, SVG images with groups  
- Built with Processing

### Web
- Go to page: [https://amelia6554.github.io/TankBattle/](https://amelia6554.github.io/TankBattle/)  
- Controls: `W`, `A`, `S`, `D` to move, LMB to shoot, `Q` to pause  
- Built with p5.js

### Mobile
- Download `.apk` file  
- Controls: joystick to move, button to shoot  
- Built with Processing Android
